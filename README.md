# 🌿 GEMPAR Config

Konfigurasi & tema untuk landing page GEMPAR yang di-host di Cloudflare Pages.

## 🚀 Quick Start

1. Edit `config/main.js` untuk ubah link, teks, fitur on/off
2. Edit `theme/variables.css` untuk ubah warna & ukuran
3. Edit `data/content/*.json` untuk ubah konten sections
4. Commit & push → GitHub Pages auto deploy

## 📁 Struktur

| Folder | Fungsi |
|--------|--------|
| `config/` | Konfigurasi JS (link, fitur, meta) |
| `theme/` | CSS variables & component styles |
| `data/content/` | Konten sections (JSON) |
| `assets/` | Gambar, icon, font |
| `docs/` | Dokumentasi |

## 🔗 URL

- Config: `https://USERNAME.github.io/gempar-config/config/main.js`
- Theme: `https://USERNAME.github.io/gempar-config/theme/variables.css`
- Assets: `https://USERNAME.github.io/gempar-config/assets/...`

## ⚠️ Ganti `USERNAME`

Cari & replace semua `USERNAME` di repo ini dengan username GitHub Anda.
